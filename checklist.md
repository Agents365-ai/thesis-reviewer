# 学位论文评审检查清单

本检查清单供 Phase 1 Step 2（逐章深入分析）使用。适用于所有学科的硕士和博士学位论文。

**使用方法：**
1. 所有论文执行本文件中的通用检查项
2. 根据学科领域，额外加载 `disciplines/` 目录下对应的学科专项检查
3. 博士论文额外执行「博士论文附加检查」部分
4. 专业学位论文额外执行「专业学位论文注意事项」部分

**学科专项检查文件：**
- `disciplines/life-sciences.md` — 生命科学（生物学、生物医学、生态学、农学等）
- `disciplines/medicine.md` — 医学与临床医学
- `disciplines/cs-ai.md` — 计算机科学与人工智能
- `disciplines/engineering.md` — 工学（机械、电气、化工、土木、材料等）
- `disciplines/chemistry.md` — 化学与材料科学
- `disciplines/physics.md` — 物理学
- `disciplines/social-sciences.md` — 人文社会科学

如论文学科不在上述列表中，仅使用本文件的通用检查项，并结合评审者的学科专业知识进行评审。

依据标准：
- GB/T 7713.1-2006《学位论文编写规则》
- GB/T 7714-2015《信息与文献 参考文献著录规则》
- GB 3100-3102-93《量和单位》
- 《中华人民共和国学位法》（2025年1月1日施行）

严重程度标记：
- 🔴 严重问题：必须修改，影响论文核心质量或违反国家标准
- 🟡 需改进：建议修改，提升论文整体水平
- 🟢 良好：该方面表现良好，值得肯定

## 维度一：学术质量（通用）

### 摘要（中英文）
- [ ] 中文摘要是否准确概括全文？是否包含目的、方法、结果、结论四要素？
- [ ] 中文摘要字数是否符合要求？（硕士约 500-800 字，博士约 1000 字）
- [ ] 硕士摘要是否突出"新见解"？博士摘要是否突出"创造性成果"？
- [ ] 英文摘要内容是否与中文摘要完全对应？语法和表达是否规范？
- [ ] 关键词 3-5 个，是否按外延层次排列（外延大的在前）？
- [ ] 中英文关键词是否一一对应？

### 绪论/文献综述
- [ ] 研究背景是否充分铺垫，让读者理解该领域的现状？
- [ ] 文献综述是否系统、全面？是否覆盖了该方向的重要文献（含经典文献和近年进展）？
- [ ] 文献综述是否有批判性分析，而非简单罗列？
- [ ] 是否清晰指出了现有研究的不足（research gap）？
- [ ] 研究问题/科学问题是否明确、聚焦？
- [ ] 研究目的和研究意义（理论意义 + 实际意义）是否有说服力？
- [ ] 技术路线或研究框架是否清晰呈现？
- [ ] 绪论末尾是否概述了论文的章节安排？

### 研究方法
- [ ] 研究方法选择是否合理，能够回答研究问题？
- [ ] 方法描述是否足够详细，能让同行重复？
- [ ] 是否说明了方法选择的理由（为什么选此方法而非其他）？
- [ ] 数据来源是否清晰、可靠？
- [ ] 分析方法是否恰当？

### 结果
- [ ] 结果呈现是否客观，与方法部分一一对应？
- [ ] 是否存在过度解读（在结果章混入讨论内容）？
- [ ] 数据是否完整？是否有明显遗漏？
- [ ] 各结果之间的逻辑关系是否清晰？
- [ ] 结果描述是否与图表数据一致？

### 讨论
- [ ] 是否将本研究结果与已有文献进行了充分对比讨论？
- [ ] 对预期外的结果是否给出了合理解释？
- [ ] 是否讨论了研究的局限性？
- [ ] 结论是否由数据充分支撑？是否存在过度推断？
- [ ] 是否提出了合理的未来研究方向？
- [ ] 是否避免了简单重复结果部分的内容？

### 结论
- [ ] 结论是否简明扼要，准确概括主要发现？
- [ ] 结论是否与研究目的对应？是否回答了绪论提出的科学问题？
- [ ] 是否避免了引入结果和讨论中未提及的新内容？
- [ ] 创新点总结是否明确、准确？

### 创新性
- [ ] 是否有明确的创新点陈述？
- [ ] 创新点是否真实成立，还是仅为增量改进？
- [ ] 创新点在领域内是否具有实际意义？
- [ ] 创新点是否与文献综述中指出的 research gap 对应？

## 维度二：写作质量（通用）

### 逻辑连贯性
- [ ] 章节间是否有清晰的逻辑链条？是否存在章节之间的断裂感？
- [ ] 各章的过渡段落是否自然衔接？
- [ ] 全文的叙事主线是否一致？
- [ ] 论文整体是否"层次分明、数据可靠、文字简练、说明透彻、推理严谨、立论正确"？（GB/T 7713.1）

### 论证严密性
- [ ] 论点-论据-论证是否完整？
- [ ] 是否存在逻辑跳跃（从前提到结论缺少推理步骤）？
- [ ] 因果关系的论述是否严谨？是否混淆了相关性与因果性？
- [ ] 是否避免了使用"文学性质的带感情色彩的非学术性词语"？（GB/T 7713.1）

### 语言表达
- [ ] 学术语言是否规范？是否存在口语化表述？
- [ ] 是否存在冗余表述或重复内容？
- [ ] 专业术语使用是否准确、全文统一？
- [ ] 句子结构是否清晰？是否存在歧义表述？
- [ ] 全文名词术语、缩略语是否前后一致？

## 维度三：格式规范（通用，依据 GB/T 7713.1-2006）

### 论文结构完整性
- [ ] 是否按顺序包含所有必要组成部分：
  1. 封面（题名、作者、导师、学位类别、专业、培养单位、日期）
  2. 独创性声明及学位论文使用授权声明（须有手写签名）
  3. 中文摘要及关键词
  4. 英文摘要及关键词
  5. 目录（编至第三级标题）
  6. 插图和附表清单（如图表较多）
  7. 符号、缩略词、术语表（如有大量缩略词）
  8. 正文（绪论/引言 → 各章 → 结论）
  9. 参考文献
  10. 附录（如需要）
  11. 致谢
  12. 作者简历及攻读学位期间发表的学术论文与研究成果
- [ ] 各部分是否另页起始？
- [ ] 各章节标题层级是否规范统一（阿拉伯数字连续标号，如 1 → 1.1 → 1.1.1）？

### 页面设置
- [ ] 纸张是否为 A4（210×297mm）？
- [ ] 页边距是否符合要求（通常上下 2.54cm，左右 3.17cm，或学校规定的 25mm）？
- [ ] 正文字体：中文宋体小四号，英文 Times New Roman？
- [ ] 正文行距：1.5 倍行距？
- [ ] 章标题：黑体（三号或四号）加粗居中？
- [ ] 节标题：黑体小四号？
- [ ] 页眉：奇数页显示章节名称，偶数页显示论文题目？
- [ ] 页码：前置部分（摘要、目录等）用罗马数字，正文起用阿拉伯数字？

### 题名（论文标题）
- [ ] 中文题名一般不超过 30 字，不设副标题？
- [ ] 题名是否准确反映论文核心内容？
- [ ] 英文题名是否与中文题名含义一致？

### 图表规范
- [ ] 所有图表是否有编号和标题（图题在图下方，表题在表上方）？
- [ ] 图表编号是否按章编排（如图 2-1、表 3-2）？
- [ ] 所有图表是否在正文中被引用？
- [ ] 图是否具有"自明性"——仅看图、图题、图例即可理解？
- [ ] 图片分辨率是否足够（≥300 dpi）？
- [ ] 表格是否使用三线表格式（顶线、底线、栏目线），避免竖线？
- [ ] 表内数据是否上下对齐？空白=未测，"—"=未发现，"0"=实测为零？
- [ ] 跨页表格是否标注"续表"并重复表头？
- [ ] 图表字号是否适中、清晰可读？
- [ ] 图表是否需要中英文双语标题（按学校要求）？

### 公式编排
- [ ] 公式是否另行居中排列？
- [ ] 公式编号格式是否正确（如 (2-1)，按章编号）？
- [ ] 公式中的变量是否在首次出现时给出定义和说明？
- [ ] 长公式转行是否在运算符（+、-、×、÷）之后？

### 参考文献（依据 GB/T 7714-2015）
- [ ] 引用格式是否全文统一（顺序编码制或著者-出版年制）？
- [ ] 是否正确使用文献类型标识（[J]期刊 [M]专著 [C]会议 [D]学位论文 [P]专利 [S]标准 [N]报纸 [DB/OL]电子资源 [R]报告）？
- [ ] 正文中的引用是否都在参考文献列表中找到？
- [ ] 参考文献列表中是否有未在正文中引用的条目？
- [ ] 参考文献是否过于陈旧？近 5 年文献占比是否合理？
- [ ] 参考文献数量是否充足？（硕士通常 ≥50 篇，博士通常 ≥150 篇）
- [ ] 每条参考文献的著录要素是否完整（作者、题名、出版信息等）？
- [ ] 参考文献排序是否正确（顺序编码制按引用顺序，著者-出版年制按姓氏排序）？

### 量和单位（依据 GB 3100-3102-93）
- [ ] 计量单位是否使用国际标准单位（SI 单位）？
- [ ] 量符号是否用斜体，单位符号是否用正体？
- [ ] 数值与单位之间是否有空格（如 25 °C 而非 25°C）？

### 其他格式
- [ ] 缩略词是否在首次出现时给出全称？
- [ ] 独创性声明是否使用学校统一格式？是否有签名和日期？
- [ ] 致谢内容是否恰当（感谢导师、实验室成员、资助项目等）？
- [ ] 作者简历是否包含教育经历和攻读学位期间的学术成果？

## 维度四：数据与结果（通用）

### 数据呈现
- [ ] 是否选择了恰当的图表类型展示数据？
- [ ] 数值是否保留了合理的有效数字位数？

### 统计分析（如涉及定量研究）
- [ ] 统计检验方法是否正确？
- [ ] 样本量是否足够支撑统计结论？
- [ ] 统计软件和版本是否注明？

### 图表质量
- [ ] 坐标轴标签是否清晰、完整（含单位）？
- [ ] 图例是否完整、位置恰当？
- [ ] 配色是否专业？

### 可重复性
- [ ] 数据来源是否可追溯？
- [ ] 数据处理流程是否透明？

---

## 维度五：学术诚信与规范（通用）

### 学术不端检查
- [ ] 是否有疑似抄袭或大段未标注引用的内容？
- [ ] 图片是否有疑似不当处理（拼接、重复使用、过度美化）？
- [ ] 数据是否有疑似造假或篡改的迹象（过于"完美"的数据、不合理的误差）？
- [ ] 一图多用：同一图片/数据是否在不同部分被重复使用却标注为不同结果？

### 引用规范
- [ ] 他人的观点和数据是否都已恰当标注引用来源？
- [ ] 大段引用（直接引文）是否有引号标注并注明出处？
- [ ] 是否存在"隐性引用"——使用了他人的观点但改写后未标注来源？

---

## 跨章检查（Phase 1 Step 2 完成后执行）

- [ ] 绪论提出的所有研究问题 → 结果和讨论中是否都给出了回应？
- [ ] 研究方法中描述的所有工作 → 结果中是否都呈现了？
- [ ] 正文中的参考文献引用 ↔ 参考文献列表是否一致？
- [ ] 图表编号是否连续？是否有跳号或重复编号？
- [ ] 缩略词表（如有）与正文使用是否一致？
- [ ] 全文中同一概念的表述是否一致（未出现同一事物多种叫法）？
- [ ] 页眉中的章节标题是否与正文章节标题一致？

## 博士论文附加检查（仅博士学位论文执行）

依据《学位法》（2025），博士学位申请人应"在学术研究领域做出创新性成果"，具有"独立从事学术研究工作的能力"，掌握"坚实全面的基础理论和系统深入的专门知识"。

### 创新性与原创贡献（博士标准）
- [ ] 创新点是否构成对领域的原创性贡献，而非仅是已有方法的应用或增量改进？
- [ ] 创新贡献是否达到可在高水平期刊/会议发表的程度？
- [ ] 是否清晰界定了本研究贡献与他人工作的边界？
- [ ] 理论/方法/发现中是否有可被同行引用和推广的核心贡献？
- [ ] 创新点是否经得起盲审专家的质疑？

### 独立研究能力
- [ ] 论文是否体现了作者独立提出问题的能力？
- [ ] 研究设计是否体现了作者独立思考和方案优化的过程？
- [ ] 面对困难或失败时，是否展示了问题诊断和方案调整的能力？
- [ ] 对结果的解读是否体现了独立的学术判断？
- [ ] 论文是否体现了"坚实全面的基础理论和系统深入的专门知识"？

### 研究体系的系统性
- [ ] 多章研究（如有）之间是否有清晰的内在逻辑联系？
- [ ] 各章研究是否围绕一个核心科学问题层层递进？
- [ ] 是否形成了一个完整的研究体系，而非多个独立小课题的拼凑？
- [ ] 总讨论/总结论是否从全局高度综合了各章发现？

### 文献综述深度（博士标准）
- [ ] 文献综述是否体现了对该领域发展脉络的全局理解？
- [ ] 是否涵盖了经典文献和最新进展？
- [ ] 是否对领域内不同学术观点和争议进行了评述？
- [ ] 综述是否自然过渡到自身研究的切入点，逻辑链条完整？
- [ ] 参考文献数量是否充足（通常博士论文 ≥150 篇）？

### 学术成果与发表
- [ ] 是否列出了在读期间的学术成果（已发表/已接收论文）？
- [ ] 已发表论文与学位论文内容的关系是否清晰说明？
- [ ] 学术成果是否满足学校/学院对博士毕业的最低发表要求？
- [ ] 如使用已发表论文作为论文章节，是否进行了必要的改写和整合（而非简单拼接）？

### 讨论与展望深度（博士标准）
- [ ] 讨论是否从更宏观的视角审视研究意义？
- [ ] 是否对研究的理论意义进行了深入阐述？
- [ ] 未来研究方向的展望是否具体、可行？
- [ ] 是否讨论了研究发现对相关领域或交叉学科的潜在影响？

## 专业学位论文注意事项

依据《学位法》（2025），专业学位侧重实践能力，论文形式可多元化。

- [ ] 论文是否体现了解决实际问题的能力（而非纯理论研究）？
- [ ] 研究课题是否来源于实际应用需求或行业问题？
- [ ] 方法和成果是否具有实际应用价值或转化前景？
- [ ] 如采用非传统论文形式（调研报告、案例分析、产品设计等），是否符合学校对专业学位论文形式的规定？
