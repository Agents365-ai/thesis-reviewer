# 硕士学位论文评审检查清单

本检查清单供 Phase 1 Step 2（逐章深入分析）使用。对每个检查项，给出严重程度标记：

- 🔴 严重问题：必须修改，影响论文核心质量
- 🟡 需改进：建议修改，提升论文整体水平
- 🟢 良好：该方面表现良好，值得肯定

---

## 维度一：学术质量

### 摘要（中英文）
- [ ] 中文摘要是否准确概括全文？是否包含目的、方法、结果、结论四要素？
- [ ] 英文摘要是否与中文摘要一致？语法和表达是否规范？
- [ ] 关键词选取是否恰当？是否覆盖论文核心概念？

### 绪论/文献综述
- [ ] 研究背景是否充分铺垫，让读者理解该领域的现状？
- [ ] 文献综述是否系统、全面？是否覆盖了该方向的重要文献？
- [ ] 文献综述是否有批判性分析，而非简单罗列？
- [ ] 是否清晰指出了现有研究的不足（research gap）？
- [ ] 研究问题/科学问题是否明确、聚焦？
- [ ] 研究目的和研究意义（理论意义 + 实际意义）是否有说服力？
- [ ] 技术路线或研究框架是否清晰呈现？

### 材料与方法
- [ ] 实验设计是否合理？是否设置了恰当的对照组？
- [ ] 生物学重复和技术重复是否充分？样本量是否足够？
- [ ] 方法描述是否足够详细，能让同行重复实验？
- [ ] 关键试剂（抗体、引物、细胞系等）是否注明了来源和批号/目录号？
- [ ] 关键仪器是否注明了型号和厂商？
- [ ] 所用软件和数据库是否注明了版本？
- [ ] 统计分析方法是否恰当？是否说明了选择该方法的理由？
- [ ] 伦理审批信息是否完整（如涉及动物实验或人体样本）？

### 结果
- [ ] 结果呈现是否客观，与方法部分一一对应？
- [ ] 是否存在过度解读（在结果章混入讨论内容）？
- [ ] 数据是否完整？是否有明显遗漏的实验结果？
- [ ] 各实验结果之间的逻辑关系是否清晰？
- [ ] 阴性结果是否如实报告？

### 讨论
- [ ] 是否将本研究结果与已有文献进行了充分对比讨论？
- [ ] 对预期外的结果是否给出了合理解释？
- [ ] 是否讨论了研究的局限性？
- [ ] 结论是否由数据充分支撑？是否存在过度推断？
- [ ] 是否提出了合理的未来研究方向？

### 结论
- [ ] 结论是否简明扼要，准确概括主要发现？
- [ ] 结论是否与研究目的对应？是否回答了绪论提出的科学问题？
- [ ] 是否避免了引入结果和讨论中未提及的新内容？

### 创新性
- [ ] 是否有明确的创新点陈述？
- [ ] 创新点是否真实成立，还是仅为增量改进？
- [ ] 创新点在领域内是否具有实际意义？

---

## 维度二：写作质量

### 逻辑连贯性
- [ ] 章节间是否有清晰的逻辑链条？是否存在章节之间的断裂感？
- [ ] 各章的过渡段落是否自然衔接？
- [ ] 全文的叙事主线是否一致？

### 论证严密性
- [ ] 论点-论据-论证是否完整？
- [ ] 是否存在逻辑跳跃（从前提到结论缺少推理步骤）？
- [ ] 因果关系的论述是否严谨？是否混淆了相关性与因果性？

### 语言表达
- [ ] 学术语言是否规范？是否存在口语化表述？
- [ ] 是否存在冗余表述或重复内容？
- [ ] 专业术语使用是否准确、一致？
- [ ] 句子结构是否清晰？是否存在歧义表述？

---

## 维度三：格式规范

### 章节完整性
- [ ] 是否包含所有必要章节：封面、摘要（中英文）、目录、绪论、材料与方法、结果、讨论、结论、参考文献、致谢？
- [ ] 是否根据需要包含附录（原始数据、补充图表等）？
- [ ] 各章节标题层级是否规范统一？

### 图表规范
- [ ] 所有图表是否有编号和标题（图在下、表在上）？
- [ ] 所有图表是否在正文中被引用？
- [ ] 图片分辨率是否足够（≥300 dpi）？
- [ ] 表格是否使用三线表格式？
- [ ] 图表的字号是否适中、清晰可读？

### 参考文献
- [ ] 引用格式是否全文统一？
- [ ] 正文中的引用是否都在参考文献列表中找到？
- [ ] 参考文献列表中是否有未在正文中引用的条目？
- [ ] 参考文献是否过于陈旧？近 5 年文献占比是否合理？
- [ ] 参考文献数量是否足够支撑文献综述？

### 其他格式
- [ ] 缩略词是否在首次出现时给出全称？
- [ ] 计量单位是否使用国际标准单位？
- [ ] 基因、蛋白质等生物学命名是否遵循规范（如基因名斜体）？

---

## 维度四：数据与结果

### 数据呈现
- [ ] 是否选择了恰当的图表类型展示数据？
- [ ] 柱状图/折线图是否包含必要的误差线？
- [ ] 是否标注了置信区间或标准差/标准误？

### 统计分析
- [ ] 统计检验方法是否正确（参数/非参数检验的选择）？
- [ ] p 值和显著性标注是否规范（如 *p<0.05, **p<0.01, ***p<0.001）？
- [ ] 样本量是否足够支撑统计结论？
- [ ] 多重比较是否进行了校正？

### 图表质量
- [ ] 坐标轴标签是否清晰、完整（含单位）？
- [ ] 图例是否完整、位置恰当？
- [ ] 配色是否专业？是否考虑了色盲友好性？
- [ ] 显微镜图片是否标注了比例尺？

### 可重复性
- [ ] 数据来源是否可追溯？
- [ ] 原始数据是否有适当的描述或附在附录中？
- [ ] 数据处理流程是否透明？

---

## 跨章检查（Phase 1 Step 2 完成后执行）

- [ ] 绪论提出的所有研究问题 → 结果和讨论中是否都给出了回应？
- [ ] 材料与方法中描述的所有实验 → 结果中是否都呈现了数据？
- [ ] 正文中的参考文献引用 ↔ 参考文献列表是否一致？
- [ ] 图表编号是否连续？是否有跳号或重复编号？
- [ ] 缩略词表（如有）与正文使用是否一致？
